Проект за другий семестр - QuickCuisine

Огляд

"QuickCuisine" - це вебсайт, який дозволяє користувачам замовляти їжу онлайн та оплачувати її через платіжну систему Liqpay. Основною метою сайту є забезпечення зручного та швидкого способу для користувачів замовляти їжу та напої, відстежувати свої замовлення та здійснювати онлайн-платежі.


Основні функції QuickCuisine:

    • Реєстрація та авторизація користувачів: Користувачі можуть зареєструвати новий обліковий запис, ввівши свій ім'я користувача, електронну пошту та пароль.
    • Авторизація через логін та пароль для доступу до персоналізованого контенту та історії замовлень.
    • Перегляд меню та додавання до кошика: Користувачі можуть переглядати доступні страви та напої різних категорій з описом, зображенням та ціною кожної страви.
    • Додавання обраних страв до кошика з можливістю змінювати кількість замовлених одиниць.
  • Оформлення замовлення: Користувачі можуть переглянути свій кошик, змінити вміст кошика, ввести адресу доставки та вибрати спосіб оплати через Liqpay чи при доставці.
  • Онлайн-оплата через Liqpay: Інтеграція з платіжною системою Liqpay для безпечної обробки онлайн-платежів.
  • Відстеження замовлень: Користувачі можуть переглядати історію своїх замовлень та відстежувати статус поточних замовлень.
  • Залишення відгуків: Користувачі можуть залишати відгуки про свої замовлення та ділитися своїм досвідом з іншими користувачами.
  • Сторінка адміністратора: Адміністратори сайту можуть переглядати та керувати замовленнями, а також редагувати страви, напої, замовленням, категоріями їжі.

Майбутній розвиток:

У майбутньому можливі покращення та розширення функціоналу QuickCuisine:

  • Інтеграція системи знижок, акцій та комбо-пропозицій для користувачів.
  • Можливість вибору інших платіжних систем для оплати замовлень.
  • Можливість налаштування індивідуальних уподобань та рекомендацій для користувачів.
  • Функція власного профілю користувача з можливістю зміни особистих даних, адреси доставки тощо.
